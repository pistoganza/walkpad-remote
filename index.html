WalkPad Remote v12

Web Bluetooth not supported.
Use Chrome on macOS, Windows, or Android.
Do NOT pair the treadmill via System Settings! Web Bluetooth handles the connection. If already paired — remove it from Bluetooth devices first.
  1. Turn on the treadmill (step on the belt), wait 5 sec
  2. Check Bluetooth on your Mac: System Settings → Bluetooth → On
  3. Click "Connect Treadmill" above
  4. In the Chrome popup select Yesoul / YS-xxx → Pair

Troubleshooting

  • No devices? — Step off the treadmill and back on to wake it up. It sleeps after 1-2 min.
  • Can't connect? — Close Yesoul Fitness / PitPat. Only one BLE connection at a time.
  • Chrome only (macOS, Windows, Android). Safari and Firefox don't support Web Bluetooth.
